About WebPicture

WebPicture is an enterprise architecture tool to build and generate DSL editors, based in EMF (Eclipse Modeling Framework) and Picture a domain specific language design to describe in an independent and simple way the graphical concepts of any metamodel representation such as nodes, relations and buttons. WebPicture takes an ecore model and a picture file to generate an online editor based in HTML 5 technologies in your internet browser.
